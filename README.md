# Portfolio Website

This is my personal portfolio website built to showcase my projects, skills, and academic achievements.

The website was developed using Next.js, TypeScript, and Tailwind CSS with a focus on responsive design and clean user experience.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Features

- Personal introduction
- Skills overview
- Project showcase
- Achievements section
- Contact information
- Responsive layout
